service accessible via :
	http://localhost:8000/cgi-bin/upload.py

## HOW TO RUN
cmd:
	cd /path/to/app/root
	python -m CGIHTTPServer 8000
browser:
	http://localhost:8000/cgi-bin/upload.py

the get_landmarks.py works as follows:
	./get_landmarks.py ../tmp
